# WHO'S THE NBA G.O.A.T. ?

Settling the G.O.A.T Debate: Who's The NBA G.O.A.T, Michael Jordan or LeBron James? 

### Getting Started

To get started, first clone this repository. Then:

```
# install dependencies
npm install
```

Then run development version:

```
npm run dev
```

To build for prod:

```
npm run build
```
